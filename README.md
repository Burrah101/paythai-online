# PayThai Online

PayThai is a foreigner-to-Thai QR payment coordination platform concept for Pattaya, Thailand.

## Local Setup

```bash
npm install
npm run dev
```

Open the local URL shown by Vite, usually:

```text
http://localhost:5173
```

## Deploy

Push this folder to GitHub and import the repo into Vercel.

## Domain

Primary domain: paythai.online
